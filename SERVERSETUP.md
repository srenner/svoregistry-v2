Server setup for deployment
===========================

* Start with new DigitalOcean droplet (Ubuntu 14.04)
* sudo apt-get update
* sudo apt-get upgrade
* apt-get install python3-pip
* pip3 install virtualenv
* mkdir svoregistry.com
* cd svoregistry.com
* virtualenv env
* source env/bin/activate
* pip3 install django
* mkdir site
*
